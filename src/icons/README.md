# Icons

Place your extension icons here:

- `icon16.png` - 16×16 pixels
- `icon32.png` - 32×32 pixels  
- `icon48.png` - 48×48 pixels
- `icon128.png` - 128×128 pixels

You can create these icons using any image editor. A simple colored circle or chat bubble icon works well for this extension.

For quick creation, you can:
1. Use an online icon generator
2. Create a simple design in any image editor
3. Export at the required sizes

The icons should be PNG format with transparency support.
